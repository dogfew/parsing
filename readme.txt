Здесь что-то вроде модуля, который будет подгружать информацию с ютуба.
Я решил использовать библиотеку YoutubeDL. По сравнению с гугловским API:

Преимущества:                           Недостатки:
Нет ограничений не использования        Довольно медленный

Последовательность работы:
1. Программа считывает каналы из channels.csv
2. Для каждого канала создаётся портфолио (create_portfolio) следующим образом:
    i. Получение ссылок на все видео на канале
    ii. Получение информации о каждом отдельном видео по ссылке.
3. Эти портфолио подгружаются в *.json файл

Для работы с большим количеством крупных каналов можно перестраховаться: аппендить к json файлу.
